# 人工智慧課程相關資訊
## 課程描述
本課程主要介紹人工智慧相關應用: 電腦視覺、自然語言處理、對話系統及Multi-Modal 對話系統並且深入介紹自然語言相關知識及演算法。從基本自然語言處理如:語料庫介紹、斷詞、停用字到深度學習相關模型:Google Word2Vector、神經網路、遞迴神經網路及最近比較流行的BERT，並以對話系統為例，介紹如何將所學運用於自然語言處理相關應用。
